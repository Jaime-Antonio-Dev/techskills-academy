# 🚀 TechSkills Academy - Plataforma de Cursos Tech

**TechSkills Academy** es una aplicación web moderna diseñada para la gestión y visualización de catálogos educativos tecnológicos. Este proyecto demuestra la integración de interfaces dinámicas con manejo de datos en formato JSON y diseño responsivo.

## 📋 Características Principales
* **Catálogo Dinámico**: Carga de cursos en tiempo real mediante `fetch` desde un archivo JSON.
* **Filtrado Inteligente**: Visualización de cursos destacados en la página de inicio y catálogo completo en la sección de cursos.
* **Experiencia de Usuario (UX)**: Ventanas modales detalladas con información técnica, iconos de progreso y cronómetros de duración.
* **Formulario de Contacto**: Sistema de validación de datos con feedback inmediato para usuarios en Bogotá y alrededores.
* **Interfaz Profesional**: Diseño basado en la identidad visual de la academia utilizando Tailwind CSS.

## 🛠️ Tecnologías Utilizadas
* **HTML5 & CSS3**: Estructura semántica y estilos base.
* **Tailwind CSS**: Framework de utilidades para un diseño moderno y rápido.
* **JavaScript (ES6+)**: Lógica de manipulación del DOM, manejo de eventos y consumo de APIs locales.
* **JSON**: Estructura de datos para el almacenamiento del catálogo de cursos.
