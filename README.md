# 🚀 TechSkills Academy - Plataforma de Cursos Tech

**TechSkills Academy** es una aplicación web moderna diseñada para la gestión y visualización de catálogos educativos tecnológicos. Este proyecto demuestra la integración de interfaces dinámicas con manejo de datos en formato JSON y diseño responsivo.

## 📋 Características Principales

Arquitectura Basada en Componentes: Interfaz modular desarrollada con Angular 17, asegurando una separación clara entre la lógica de negocio y la presentación.

Gestión Asíncrona de Datos: Implementación de Servicios (Injectables) para el consumo de datos desde archivos JSON, optimizando la carga de memoria mediante el uso de interfaces de TypeScript.

Navegación SPA (Single Page Application): Uso de Angular Router para una navegación fluida entre el inicio y el catálogo, eliminando las recargas de página innecesarias.

Experiencia de Usuario Dinámica (UX): Visualización de cursos a través de componentes reutilizables, con modales informativos integrados para detalles técnicos y métricas de progreso.

Diseño Responsivo y Moderno: Estilizado mediante Tailwind CSS, garantizando una visualización óptima en dispositivos móviles y de escritorio.

Validación de Formularios: Sistema de contacto con validaciones en tiempo real para asegurar la integridad de la información recolectada.


## 🛠️ Tecnologías Utilizadas

Angular 17: Framework principal para la arquitectura SPA (Single Page Application).

TypeScript: Lenguaje base para asegurar un tipado fuerte y código mantenible.

Tailwind CSS: Diseño responsivo y estilizado basado en utilidades.

RxJS & Services: Manejo de datos asíncronos y lógica de negocio centralizada.

JSON: Almacenamiento local de la base de datos de cursos.
